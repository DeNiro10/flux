## Iniciar o Projeto

Execute apenas:

```bash
./INICIO_SIMPLES.sh
```

**Pronto!** O aplicativo vai abrir no navegador automaticamente.

## Ou manualmente:

```bash
npm run dev
```

## Acessar

- **Frontend:** http://localhost:5173
- **Backend:** http://localhost:3000/api

## Parar

Pressione `Ctrl+C` no terminal.

## Primeira Vez?

Se for a primeira vez, execute:

```bash
npm install
./INICIO_SIMPLES.sh
```

## Problemas?

Se der erro de `better-sqlite3`, execute:

```bash
source ~/.nvm/nvm.sh
nvm use 24
npm rebuild better-sqlite3
npm run dev
```
